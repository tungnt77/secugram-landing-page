# Secugram Landing Page

Static Secugram marketing landing page prototype.

Open `ui_kits/website/index.html` in a browser to view the site.

## Build

The page ships with precompiled React JSX in `ui_kits/website/dist/site.js`.

After editing any JSX source in `ui_kits/website`, rebuild with:

```bash
node tools/build-website.mjs
```
