# Secugram Website UI Kit

## Overview
High-fidelity interactive recreation of the Secugram marketing landing page.
Based on: `assets/website-header-reference.png` and brand guidelines.

## Design Width
**1440px** (scales responsively down to 1024px)

## Sections / Components
- **Header.jsx** — Top nav bar + hero split section
- **Features.jsx** — Services / feature cards grid
- **Pricing.jsx** — Pricing tiers section
- **Footer.jsx** — Footer with links and branding

## Usage
Open `index.html` for the full interactive landing page prototype.
Components are modular JSX files loaded via Babel.

## Navigation Flow
- Home (hero) → Pricing section (smooth scroll on "See Pricing" CTA)
- Contact section at bottom of page
